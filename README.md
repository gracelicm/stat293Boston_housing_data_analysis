# stat293Boston_housing_data_analysis
This is the GitHub repository created for the project analyzes Boston housing data for course STAT293. The goal if this project is to examines the determinants of housing prices in the Boston area using an extended version of the Boston Housing dataset, with corrected median home value (cmedv) as the primary response variable.

edacode.R is the R code for exploratory data analysis.

data_analysis.Rmd is the R code for multiple imputation of missing data, fitting the Spatial Autoregressive (SAR) model, and conducting sensitivity analysis.

data_analysis.pdf is the output generated from data_analysis.Rmd.

Moran_test.R is the R code for Spatial visualization and Moran’s I statistic.

housing.rdata is the dataset used in this project.

Housing_data_description.pdf is the data description for the dataset used here.
